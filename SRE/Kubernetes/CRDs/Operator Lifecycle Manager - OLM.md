> OLM is a component of the Operator Framework - an open source toolkit to manage Kubernetes native applications (i.e. Operators) - in an effective, automated, and scalable way.

# 1. Install OLM on K8s

^f56b44

To install OLM on K8s, run:

```shell
curl -sL https://github.com/operator-framework/operator-lifecycle-manager/releases/download/v0.28.0/install.sh | bash -s v0.28.0
```
